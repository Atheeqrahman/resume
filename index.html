My courses

Google Data Analytics Professional Certificate

The Google Data Analytics course on Coursera is a comprehensive program designed to help individuals gain foundational knowledge and skills in data analysis. The course covers essential concepts such as data collection, cleaning, analysis, visualization, and presentation. It also provides hands-on experience using tools like Google Sheets, Google Analytics, and SQL. The course consists of both theoretical and practical components, with real-world case studies to reinforce learning. Upon completion of the course, individuals will have the necessary skills and knowledge to enter the field of data analytics or further their career in the industry.

IBM Data Science Professional Certificate

The IBM Data Science course on Coursera is a comprehensive program designed to equip learners with the knowledge and skills required to become successful data scientists. The course covers a range of topics, including data analysis, machine learning, data visualization, and data ethics. The curriculum is hands-on, with learners working on real-world projects using industry-standard tools such as Jupyter notebooks and IBM Watson Studio. Upon completion of the course, learners will have a solid foundation in data science and be able to apply their skills to solve complex business problems. The course is suitable for beginners as well as experienced professionals looking to upskill or transition to a career in data science.

My projects

Dog vs Cat Image Classification using Convolutional Neural Networks

The project trains a Convolutional Neural Network (CNN) model using the Dogs vs Cats dataset from Kaggle to classify images of dogs and cats. The CNN model has multiple convolutional and pooling layers, followed by fully connected layers, and uses binary cross-entropy loss with Adam optimizer.

Personal Finance Analysis Dashboard

This project is aimed at analyzing personal finance data using Power BI to provide a comprehensive dashboard view of an individual's financial situation. The dashboard displays various financial metrics such as income, expenses, savings, and investment performance. It also provides insights into spending patterns, helps in identifying areas of overspending and underspending, and recommends actionable steps to improve financial health. The project enables users to monitor their financial progress, make informed decisions, and achieve their financial goals.
